# oliststreaming Container (Streaming_Data)

This container stores data for the streaming pipeline.

-----------------------------------------------------------------------------------------------------------------------------------------------------------------
## Data Flow

- Source CSV files are read by a Python producer.
- The producer loads the files into DataFrames.
- The records are converted to JSON events.
- The JSON events are sent to Azure Event Hubs.
- Databricks Structured Streaming consumes the events from Event Hubs.
- Data is stored in the Bronze Layer, transformed in the Silver Layer, and later processed by the shared Gold Layer notebook used by both the batch and streaming pipelines.

-----------------------------------------------------------------------------------------------------------------------------------------------------------------
## Folders

- Bronze_Layer - Raw streaming data stored as Delta tables.
- Silver_Layer - Cleaned and validated streaming data stored as Delta tables.
- Checkpoints - Streaming checkpoint data for recovery and incremental processing.

## Note

***The Gold Layer is generated by the same Gold notebook used by both the batch and streaming pipelines, ensuring a unified business-ready data model.***
